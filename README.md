### Features

- JWT based authentication
- Protect route 

### Install

    npm install react-jwt-auth

### Quickstart

```jsx
import { setJWTtoken } from 'react-jwt-auth';


const jwt_token = 'jsjdjdjsxxfd'
setJWTtoken(jwt_token)

```
